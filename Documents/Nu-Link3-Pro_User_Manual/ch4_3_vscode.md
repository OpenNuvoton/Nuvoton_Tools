# 4.3 Visual Studio Code

The Nuvoton NuMicro Cortex-M Pack serves as a comprehensive development toolkit tailored for Nuvoton's NuMicro Cortex-M microcontrollers within the Visual Studio Code environment.

---

## Features

- **Essential Extensions:** The pack bundles all necessary Visual Studio Code extensions to support embedded development.
- **Configuration and SVD Support:** Built-in configuration files and System View Description (SVD) support enable register-level debugging and more precise control over microcontroller operations.
- **OpenOCD/pyOCD Integration:** The toolkit provides automated setup for both OpenOCD-based and pyOCD-based debugging workflows.
- **Debug Tools:** Included PinView and LCDView extensions facilitate peripheral debugging.
- **Sample Code:** The pack is compatible with NuMicro BSP projects, allowing for a faster and more efficient development start.

---

## Getting Started

To begin working with the Nuvoton NuMicro Cortex-M Pack:

1. **Install the Extension Pack:**
   - Open Visual Studio Code.
   - Navigate to the Extensions marketplace (Ctrl+Shift+X).
   - Search for **"Nuvoton NuMicro Cortex-M Pack"** or visit the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Nuvoton.nuvoton-numicro-cortex-m-pack).
   - Click **Install** to add the pack to your environment.

2. **Access Getting Started Guide:**
   - After installation, the Getting Started section will be available in the Visual Studio Code welcome page.
   - Follow the setup instructions to configure OpenOCD/pyOCD and other necessary tools.

3. **Open NuMicro BSP Projects:**
   - The pack is compatible with NuMicro Board Support Package (BSP) projects.
   - Open your existing BSP project or create a new one to leverage the pack's features.
